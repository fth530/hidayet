Build a mobile app called "Hidayet" using React Native 
and Expo. This is a step-by-step Islamic guide for new 
Muslims (converts/reverts). The tone is warm, welcoming, 
patient, and never judgmental.

DESIGN DIRECTION:
  Spiritual, calm, premium Islamic aesthetic.
  Think: Calm app meets Islamic art.

    border: rgba(255, 255, 255, 0.08), rounded 20px
