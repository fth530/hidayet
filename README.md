## 🚀 Hakkında

**Hidayet**, ibadetlerinizi, dini öğreniminizi ve günlük hedeflerinizi takip etmenizi sağlayan şık bir mobil uygulamadır. Göz yormayan koyu lacivert renk teması ve tamamen internetsiz (çevrimdışı) çalışabilen altyapısıyla her zaman ve her yerde yanınızdadır.

## ✨ Temel Özellikler

- 🌙 **İnternetsiz Namaz Vakitleri:** Diyanet hesaplama yöntemine uygun matematiksel algoritma ile, internet bağlantısına ihtiyaç duymadan namaz vakitlerini cihazınızda anında hesaplar.
- 📿 **30 Günlük Gelişim Takibi:** Dini bilgilerinizi her gün bir adım ileri taşıyın. Öğrendiğiniz günleri işaretleyin, aralıksız çalışma (streak) serileri yapın ve seviyenizi yükseltin (Temel, Orta, İleri).
- 🕋 **Akıllı Zikirmatik:** Günlük çekmeniz gereken *Sübhanallah*, *Elhamdülillah* ve *Allahu Ekber* zikirlerini takip eder. Uygulamayı kapatsanız bile kaldığınız otomatik kaydedilir.
- 🥇 **Modern ve Akıcı Tasarım:** Karmaşadan uzak, premium bir arayüz. Butonlardaki gerçek dokunma hissiyatı (haptic) ve pürüzsüz animasyonlarla kullanım kolaylığı sunar.

## 🛠️ Nasıl Çalıştırılır?

Projeyi bilgisayarınıza indirip kolayca başlatmak için aşağıdaki adımları izleyebilirsiniz:

```bash
# 1. Projeyi bilgisayarınıza indirin
git clone https://github.com/fth530/hidayet.git

# 2. Gerekli kütüphaneleri yükleyin
pnpm install

# 3. Uygulamayı başlatın
cd artifacts/hidayet
pnpm dev
# (veya sadece: npx expo start)
```

> **Not:** Telefonunuzda uygulamayı canlı olarak görmek için, uygulama mağazasından **Expo Go** uygulamasını indirip açılan terminaldeki QR kodunu okutmanız yeterlidir.
